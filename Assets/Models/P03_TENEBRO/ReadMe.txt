・Sceneデータ「DEMO_MagicaCloth」を動作させるには、アセット「Magica Cloth」が必要になります。
また、「Magica Cloth（マギカ・クロス）」を動作させるにはUnity Job System + Burstコンパイラが必要になります。
アセットストア
https://assetstore.unity.com/packages/tools/physics/magica-cloth-160144

・Sceneデータ「DEMO_DynamicBone」を動作させるには、アセット「Dynamic Bone」が必要になります。
アセットストア：
https://assetstore.unity.com/packages/tools/animation/dynamic-bone-16743

・ヒューマノイドキャラクターのマテリアルはフォルダ「Mterials」内の
パッケージデータ「URPconvert」を展開するこでURP対応マテリアルへと上書き変換できます。


-The asset "Magica Cloth" is required to operate the Scene data "DEMO_Magica Cloth".
In addition, the Unity Job System + Burst compiler is required to operate "Magica Cloth".
Asset store
https://assetstore.unity.com/packages/tools/physics/magica-cloth-160144

-The asset "Dynamic Bone" is required to operate the Scene data "DEMO_Dynamic Bone".
Asset store:
https://assetstore.unity.com/packages/tools/animation/dynamic-bone-16743

-Humanoid character materials are in the folder "Mterials"
By expanding the package data "URP convert", it can be overwritten and converted to URP compatible material.
